# SampleMosbyMVP
Simple example of implementing clean architecture in Android app. Presentation layer implements MVP with Mosby.

## Requirements
* Android Studio 3.5 Canary 5
* Kotlin 1.3.21
* Android Gradle Plugin 3.5.0-alpha05
* Gradle wrapper 5.1.1
* AAPT 2
